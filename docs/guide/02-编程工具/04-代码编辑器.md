# 代码编辑器

## 代码编辑器的重要性

在 Vibe Coding 过程中，一个好的代码编辑器可以显著提升你的效率和体验。虽然 AI 可以帮你生成代码，但你仍然需要阅读、编辑和管理代码。因此，选择和配置一个合适的代码编辑器是非常重要的。

好的代码编辑器应该具备以下特点：语法高亮，让代码更易读；智能补全，减少输入量；错误提示，及时发现问题；版本集成，方便管理代码；扩展性强，可以添加各种功能。

## VS Code：首选编辑器

VS Code（Visual Studio Code）是目前最流行的代码编辑器，也是 Vibe Coding 的首选工具。

### 为什么选择 VS Code

VS Code 有很多优势让它成为 Vibe Coding 的最佳选择：

免费开源，完全免费使用，没有版权问题。跨平台，支持 Windows、Mac、Linux 三大系统。性能优秀，启动快，运行流畅。生态丰富，有海量的扩展程序可供选择。AI 集成，内置 GitHub Copilot 支持，也支持其他 AI 扩展。

### 必备扩展程序

以下是 Vibe Coding 推荐安装的 VS Code 扩展：

**GitHub Copilot**。这是 GitHub 推出的 AI 编程助手，可以提供实时的代码建议。它和 Vibe Coding 的理念很契合，AI 可以在你编写代码时给出建议。

**GitHub Copilot Chat**。Copilot 的对话版本，允许你通过对话的方式获取代码帮助。它可以解释代码、生成新代码、修复 bug 等。

**Prettier**。代码格式化工具，可以自动调整代码格式，让代码保持一致的风格。保存文件时自动格式化，不需要手动操作。

**ESLint / TSLint**。代码检查工具，可以发现代码中的问题并提供修复建议。它可以帮助你写出更高质量的代码。

**Auto Rename Tag**。自动重命名标签，当修改 HTML 或 JSX 标签时，自动配对修改对应的闭合标签。

**CSS Peek**。查看 CSS 定义，当你在 HTML 中选中一个类名时，可以快速跳转到对应的 CSS 定义。

**Live Server**。本地开发服务器，可以快速启动一个本地服务器预览网页。对于前端开发非常方便。

### VS Code 配置优化

除了安装扩展，还需要对 VS Code 进行一些配置优化。

打开设置（Ctrl+,），进行以下配置：

**Editor: Format On Save**。启用保存时自动格式化，勾选这个选项可以让代码每次保存时自动格式化。

**Editor: Tab Size**。设置缩进大小，建议设置为 4 或 2，根据项目规范来定。

**Editor: Word Wrap**。设置代码换行，可以选择 on 或 off，根据个人习惯。

**Files: Auto Save**。设置自动保存，可以选择 afterDelay 并设置延迟时间。

### 快捷键提升效率

熟练使用快捷键可以大大提高效率。以下是一些最常用的快捷键：

Ctrl+P 快速打开文件。Ctrl+Shift+P 打开命令面板。Ctrl+Shift+L 选择所有匹配项。Ctrl+D 选择下一个匹配项。Alt+Up/Down 移动行。Ctrl+Shift+K 删除行。

建议花一些时间学习快捷键，长期来看可以节省大量时间。

## 其他代码编辑器选择

虽然 VS Code 是推荐的首选，但也有一些其他选择值得了解。

### Zed

Zed 是一个高性能的代码编辑器，用 Rust 编写，运行速度非常快。它支持多人协作，界面简洁现代。目前 Zed 还在快速发展中，可以关注后续版本。

### Neovim

Neovim 是 Vim 的现代复刻版，扩展性极强，深受资深开发者喜爱。但学习曲线陡峭，需要投入大量时间学习。如果你是 Vim 爱好者，Neovim 是很好的选择。

### Sublime Text

Sublime Text 是一个轻量级但功能强大的编辑器。它的响应速度很快，界面简洁。但它不是免费的，需要购买许可证。

### 在线编辑器

对于简单的 Vibe Coding 任务，在线编辑器也是不错的选择：

StackBlitz 提供完整的 Node.js 开发环境，可以直接在浏览器中开发。CodeSandbox 类似，支持多种框架。GitHub Codespaces 集成在 GitHub 中，可以直接在仓库中开发。

## 编辑器与 AI 工具的集成

现代代码编辑器都可以和 AI 工具深度集成。

### 内置 AI 助手

VS Code 的 GitHub Copilot 是集成度最高的方案。安装 Copilot 扩展后，你可以在编写代码时获得实时的 AI 建议。Copilot 可以根据上下文预测你想写的代码，按 Tab 键即可接受建议。

### 聊天式 AI

除了代码补全，很多编辑器还支持聊天式 AI 功能。你可以在编辑器中打开一个聊天窗口，用自然语言描述需求，AI 会生成相应的代码。

GitHub Copilot Chat、Continue、Cline 等都是这类工具的代表。

### 项目级 AI

对于复杂项目，你可能需要项目级的 AI 支持。这类工具可以理解整个项目的结构和上下文，提供更准确的建议。

**Cursor** 是目前最流行的 AI 优先代码编辑器，基于 VS Code 分支开发。它的特点是：

- 支持多种 LLM：GPT-4o、Claude 3.5/3.7/4.0、Gemini 2.5 等
- **Composer 模式**：可以进行多文件编辑和重构
- **Agent 模式**：可以探索仓库、阅读文档、浏览网页、执行终端命令，但每次更改都需确认
- 上下文控制：使用 `@codebase` 和 `@files` 标签精确指定上下文
- 保持 VS Code 熟悉的界面，可直接迁移主题、快捷键和扩展
- 定价：$20/月 Pro 版，免费版每月 2000 次补全

**Windsurf**（原 Codeium，2025 年 1 月更名，后被 OpenAI 收购）是首个「智能体原生 IDE」：

- **Cascade 智能体**：自动扫描整个仓库、选择相关文件、执行测试和命令、直接修改代码，无需确认对话框
- 自动上下文管理：使用全仓库嵌入，Cascade 自动选择相关上下文
- 自研 SWE-1 模型：提供超低延迟的内联补全
- 支持多种 IDE：VS Code、IntelliJ、JetBrains 系列
- 定价：$15/月 Pro 版，比 Cursor 更实惠
- 适合大型复杂代码库和团队协作

## 编辑器选择建议

对于 Vibe Coding 学习者，我们给出以下建议：

**完全新手**：从 VS Code 开始。它功能完善，学习资源丰富，可以满足你所有的需求。

**想要更强 AI 能力**：尝试 Cursor。它保持了 VS Code 的熟悉感，同时提供了更强大的 AI 编程能力，适合想要提升效率的开发者。

**追求自动化和团队协作**：考虑 Windsurf。它的 Cascade 智能体可以自动处理更多任务，适合大型项目和团队协作场景。

**预算有限**：Windsurf 的 $15/月 价格更亲民，或者使用 VS Code + GitHub Copilot 的免费额度。

不要花太多时间在配置编辑器上。够用就好，把时间花在真正重要的编程学习上。

尝试不同的工具，但要保持开放的心态。找到最适合自己的工具组合，然后深入掌握它。
