# 知识百科

## 计算机基础

### 程序是什么

程序是一系列指令的集合，告诉计算机做什么。程序可以用各种编程语言编写，最终会被编译或解释成计算机能够执行的机器码。

### 编译器与解释器

**编译器**将整个源代码一次性翻译成目标代码（如 .exe 文件），翻译过程较长，但运行时不需要再次翻译。

**解释器**逐行读取源代码并立即执行，不需要事先编译，运行时每次都需要解释。

### 二进制与编码

计算机内部使用二进制（0 和 1）存储和处理数据。字符编码（如 UTF-8）将文字转换为二进制，以便计算机处理和存储。

### 内存与存储

**内存**（RAM）是临时存储，存取速度快，但断电后数据丢失。**存储**（硬盘、SSD）是永久存储，存取速度较慢，但断电后数据保留。

## Web 基础

### URL 解析

URL（统一资源定位符）用于定位网络上的资源。格式：协议://域名:端口/路径?查询参数#片段。例如：https://www.example.com:443/search?q=hello#results

### HTTP 协议

HTTP 是 Web 通信的基础协议。**请求方法**：GET（获取资源）、POST（提交数据）、PUT（更新资源）、DELETE（删除资源）。

**状态码**：200 成功、404 未找到、500 服务器错误等。

**请求头**：Content-Type、Authorization、User-Agent 等。

**请求体**：POST/PUT 请求中发送的数据。

### 浏览器工作原理

浏览器的基本工作流程：输入 URL 后，浏览器向服务器发送请求。服务器返回 HTML、CSS、JavaScript 等资源。浏览器解析和渲染页面。执行 JavaScript 脚本。显示最终页面。

### 前端与后端

**前端**是用户直接看到的部分，包括 HTML、CSS、JavaScript。**后端**是服务器端，处理业务逻辑、数据库操作等。前后端通过 API 进行通信。

## 编程语言基础

### 变量与数据类型

**变量**是存储数据的容器。**数据类型**定义数据的种类：字符串（String）、数字（Number）、布尔值（Boolean）、数组（Array）、对象（Object）等。

### 运算符

**算术运算符**：+、-、*、/、%（取余）。

**比较运算符**：==（等于）、===（严格等于）、!=、>、<、>=、<=。

**逻辑运算符**：&&（与）、||（或）、!（非）。

### 条件判断

根据条件执行不同的代码：if...else、switch...case、三元运算符。

### 循环

重复执行代码：for 循环、while 循环、for...in、for...of。

### 函数

封装可重复使用的代码块。函数可以有参数（输入）和返回值（输出）。

### 面向对象

**类**是对象的模板，定义对象的属性和方法。**对象**是类的实例。**继承**允许一个类使用另一个类的属性和方法。**封装**隐藏内部实现，只暴露接口。

## 数据结构

### 数组

有序的数据集合，支持索引访问和多种操作方法。

### 对象/字典

键值对形式的数据结构，支持通过键快速查找值。

### 栈（LIFO）

后进先出的数据结构，只允许在栈顶进行插入和删除。

### 队列（FIFO）

先进先出的数据结构，允许在队尾插入，队头删除。

### 链表

由节点组成的数据结构，每个节点包含数据和指向下一个节点的指针。

### 树

层次化的数据结构，每个节点可以有多个子节点。二叉树是最常见的树结构。

### 图

由节点（顶点）和边组成的数据结构，用于表示复杂的关系。

## 算法基础

### 时间复杂度

衡量算法执行时间随数据量增长的增长率。常见复杂度：O(1)、O(log n)、O(n)、O(n log n)、O(n²)。

### 排序算法

**冒泡排序**：相邻元素比较交换，时间复杂度 O(n²)。

**快速排序**：选择基准，分区递归，时间复杂度 O(n log n)。

**归并排序**：分治思想，合并有序数组，时间复杂度 O(n log n)。

### 搜索算法

**线性搜索**：逐个检查，时间复杂度 O(n)。

**二分搜索**：在有序数组中折半查找，时间复杂度 O(log n)。

### 递归与迭代

**递归**是函数调用自身，需要有终止条件。**迭代**是使用循环重复执行。两者通常可以互相转换。

## 数据库基础

### SQL 基础

**SELECT**：查询数据。**INSERT**：插入数据。**UPDATE**：更新数据。**DELETE**：删除数据。

### 表设计

**主键**：唯一标识每条记录的字段。**外键**：关联其他表的字段。**索引**：提高查询速度的数据结构。

### 数据库范式

**第一范式（1NF）**：每个字段都是原子的。**第二范式（2NF）**：满足 1NF，且非主键字段完全依赖主键。**第三范式（3NF）**：满足 2NF，且非主键字段不传递依赖主键。

### 事务

数据库事务的 ACID 特性：原子性（Atomicity）、一致性（Consistency）、隔离性（Isolation）、持久性（Durability）。

## 版本控制

### Git 基础

**git init**：初始化仓库。**git add**：添加到暂存区。**git commit**：提交。**git push**：推送到远程。**git pull**：拉取远程更新。**git branch**：管理分支。**git merge**：合并分支。

### 分支策略

常用的分支策略：main/master（主分支）、develop（开发分支）、feature（功能分支）、hotfix（紧急修复分支）。

### 合并请求

通过合并请求（Merge Request / Pull Request）进行代码审查和协作。

## 开发工具

### IDE 与编辑器

**IDE**（集成开发环境）是功能全面的开发工具，如 VS Code、WebStorm。**编辑器**是轻量级的代码编辑工具，如 Vim、Sublime Text。

### 包管理器

**npm**（Node Package Manager）：Node.js 的包管理器。**pnpm**：更高效的包管理器。**pip**：Python 的包管理器。**cargo**：Rust 的包管理器。

### 构建工具

**Vite**：现代前端构建工具。**Webpack**：功能强大的打包工具。**Gulp**：任务自动化工具。**Babel**：JavaScript 编译器。

### 测试工具

**Jest**：JavaScript 测试框架。**Vitest**：Vite 的测试框架。**Cypress**：端到端测试工具。**Playwright**：浏览器自动化测试工具。
