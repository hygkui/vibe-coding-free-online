# 简历模板

## 简历的重要性

简历是求职的第一关。一份好的简历可以帮你获得面试机会。对于 Vibe Coding 学习者来说，如何展示你的技能和项目经验尤为重要。

## 简历结构

### 基本信息

**姓名**：全名，突出显示。

**联系方式**：电话、邮箱、LinkedIn、个人网站。

**求职意向**：目标岗位、期望城市、到岗时间。

### 教育背景

学校名称、专业、学位、在读时间。相关课程（针对应届生）。GPA（如果优秀）。

### 技能清单

**技术栈**：列出你熟悉的技术，按熟练程度排序。

**语言能力**：英语水平、其他语言。

**其他技能**：项目管理、工具使用等。

### 项目经验

这是简历的核心部分。每 个项目都要包含：项目名称、你的角色、项目时间、技术栈、项目描述、你的贡献。

### 工作/实习经历

公司名称、职位、工作时间、工作内容、主要成果。

### 其他（可选）

开源贡献、技术博客、获奖经历、兴趣爱好。

**项目描述**：开发了一个 Chrome 浏览器扩展，帮助用户管理和追踪待办事项。

**主要贡献**：
- 设计并实现插件界面，使用原生 Web 技术
- 实现数据本地存储和跨设备同步

## 简历优化建议

### 量化成果

用数字说话，让成果更具体：

**优化前**："优化了网站性能"

**优化后**："优化页面加载速度，通过代码分割和懒加载将首屏时间从 3s 降低到 1.2s"

### 突出亮点

找出你最突出的特点，在简历中突出展示：技术亮点、项目亮点、成果亮点。

### 简洁有力

简历控制在 1-2 页。每句话都要有价值。删除与岗位无关的内容。

### 针对投递

针对不同公司和岗位，调整简历重点。了解公司的技术栈，在技能清单中突出相关技能。

### 持续更新

定期更新简历，添加新项目和技能。记录项目中的关键数据和成果。

## 面试准备

### 技术准备

复习基础知识：数据结构、算法、计算机网络等。准备编程题的练习。了解应聘岗位所需的技术栈。

### 项目准备

熟悉简历上的每个项目。准备项目相关的深入问题。思考项目的改进点和挑战。

### 行为面试

准备常见行为面试问题。使用 STAR 法则组织答案。准备几个展现团队协作和问题解决的例子。
