# AI 技术

## AI 技术概述

了解 AI 技术的基本原理，可以帮助你更好地使用 AI 工具，理解它们的能力和局限性。

## 机器学习基础

### 什么是机器学习

机器学习是人工智能的一个分支，让计算机通过数据学习，而不是通过明确的编程来完成任务。机器学习系统从数据中学习模式，然后用学到的模式进行预测或决策。

### 机器学习的类型

**监督学习**：使用标注的数据进行训练，每个输入都有对应的正确输出。应用：分类、回归。

**无监督学习**：使用没有标注的数据，发现数据中的模式。应用：聚类、降维。

**强化学习**：通过与环境交互学习，根据奖励信号调整行为。应用：游戏、机器人控制。

### 深度学习

深度学习是机器学习的一个分支，使用多层神经网络。深度学习在图像识别、自然语言处理等领域取得了突破性进展。

## 大语言模型

### 什么是大语言模型

大语言模型（LLM）是一种基于深度学习的语言模型，通过在海量文本数据上训练，学习语言的模式和知识。GPT、Claude、Llama 等都是大语言模型。

### 大语言模型的工作原理

大语言模型的核心是 Transformer 架构。它通过注意力机制理解输入文本的上下文关系，预测下一个最可能出现的词或生成完整的回答。

### 大语言模型的能力

**文本生成**：生成文章、代码、对话等。

**知识问答**：回答各种问题。

**代码编写**：生成和解释代码。

**翻译**：多语言翻译。

**摘要**：生成长文本的摘要。

**推理**：进行逻辑推理和分析。

### 大语言模型的局限性

**知识截止日期**：模型的知识有时间限制，可能不了解最新信息。

**幻觉**：可能生成看似合理但实际错误的信息。

**上下文限制**：单次对话有长度限制。

**推理错误**：复杂的推理可能出错。

## 提示工程

### 什么是提示工程

提示工程是设计、优化输入提示，以获得更好输出的实践。提示的质量直接影响模型的输出质量。

### 提示的基本要素

**指令**：明确告诉模型要做什么。

**上下文**：提供背景信息。

**输入数据**：模型需要处理的具体内容。

**输出格式**：期望的输出形式。

### 提示技巧

**明确具体**：指令要清晰具体。

**提供示例**：给出发示例可以帮助模型理解。

**分解任务**：将复杂任务分解成简单步骤。

**使用系统提示**：设置模型的角色和行为规范。

**迭代优化**：根据输出不断调整提示。

## AI 在编程中的应用

### 代码生成

AI 可以根据自然语言描述生成代码。这是 Vibe Coding 的核心能力。AI 生成的代码需要人类审查和验证。

### 代码补全

AI 可以在编写代码时提供补全建议。GitHub Copilot 是这类工具的代表。

### 代码解释

AI 可以解释代码的功能和工作原理。这对于学习他人代码很有帮助。

### 代码审查

AI 可以进行初步的代码审查，发现潜在的问题和改进点。

### Bug 修复

AI 可以帮助分析和修复代码中的 bug。通过描述错误信息，AI 可能提供解决方案。

## AI API 服务

### OpenAI API

提供 GPT 系列模型的 API 服务。可以调用 GPT-4o、o1、o3 等模型。计费方式按 token 使用量，GPT-4o 约 $5/百万输入 token。

### Anthropic API

提供 Claude 模型的 API 服务。Claude 3.5 Sonnet 在编程任务上表现出色，$3/百万输入 token。

### 其他 API

**Google Gemini API**：Google 的 AI 模型 API，有免费额度。

**国内 AI API**：通义千问、文心一言、Kimi、月之暗面、DeepSeek 等，对中文支持更好。

## AI 应用的架构

### 提示链

将复杂任务分解成多个步骤，每个步骤调用一次 AI。中间步骤的输出作为下一步骤的输入。

### RAG（检索增强生成）

结合检索和生成。先从知识库中检索相关信息，然后将检索结果和用户问题一起发送给 AI，让 AI 基于检索结果回答。

### 代理（Agent）

AI 代理可以自主规划和执行任务。代理可以调用工具、执行多步骤任务。适合复杂任务的自动化。

## AI 伦理与安全

### 使用限制

尊重 AI 服务提供商的使用条款。注意 AI 生成内容的版权问题。不要使用 AI 生成有害内容。

### 安全考虑

不要在 AI 对话中输入敏感信息。注意 AI 生成代码的安全性。验证 AI 生成的内容。

### 负责任使用

AI 是辅助工具，不是替代品。保持批判性思维，不过度依赖 AI。注明 AI 辅助生成的内容。
