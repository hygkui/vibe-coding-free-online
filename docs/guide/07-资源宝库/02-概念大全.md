# 概念大全

## 编程基础概念

### 编程范式

**面向过程编程**：按步骤执行，关注过程和函数。

**面向对象编程**：把事物抽象成对象，关注类和对象。

**函数式编程**：把计算当作函数求值，避免状态变化。

**响应式编程**：数据流和变化传播的编程范式。

### 开发模式

**敏捷开发**：迭代式开发，持续交付。

**瀑布模型**：线性顺序开发，文档驱动。

**DevOps**：开发与运维结合，自动化流程。

**TDD（测试驱动开发）**：先写测试，再写代码。

## Web 开发概念

### 前端概念

**SPA（单页应用）**：整个应用在一个页面，通过 JS 动态更新。

**SSR（服务端渲染）**：在服务器渲染页面，提升首屏速度。

**SSG（静态站点生成）**：在构建时生成静态 HTML。

**CSR（客户端渲染）**：在浏览器中渲染页面。

**PWA（渐进式 Web 应用）**：可以像原生应用一样使用的 Web 应用。

**SEO（搜索引擎优化）**：提高网站在搜索引擎中的排名。

**SSR vs CSR**：服务端渲染 vs 客户端渲染的对比。

### 后端概念

**API（应用程序接口）**：程序之间通信的接口。

**RESTful API**：基于 HTTP 的 REST 架构风格的 API。

**GraphQL**：客户端指定所需数据的查询语言。

**WebSocket**：双向实时通信协议。

**RPC（远程过程调用）**：调用远程服务器上的函数。

**中间件**：处理请求和响应的软件层。

### 数据库概念

**ACID**：数据库事务的四个特性（原子性、一致性、隔离性、持久性）。

**CRUD**：创建、读取、更新、删除四种基本操作。

**索引**：提高查询速度的数据库结构。

**事务**：一组原子性的数据库操作。

**ORM（对象关系映射）**：把数据库操作映射为对象操作。

## 云计算概念

### 部署模式

**IaaS（基础设施即服务）**：提供虚拟机、存储等基础设施。

**PaaS（平台即服务）**：提供应用开发和部署平台。

**SaaS（软件即服务）**：提供可直接使用的软件。

**FaaS（函数即服务）**：无服务器计算，按调用付费。

### 容器化

**Docker**：应用容器化平台。

**容器**：轻量级的虚拟化技术。

**镜像**：容器的只读模板。

**Kubernetes**：容器编排平台，管理容器集群。

### 云服务

**CDN（内容分发网络）**：分布式服务器，加速内容分发。

**负载均衡**：分发流量到多个服务器。

**自动扩展**：根据负载自动调整资源。

## AI 概念

### 机器学习

**监督学习**：使用标注数据训练模型。

**无监督学习**：从无标注数据中发现模式。

**强化学习**：通过与环境交互学习。

**深度学习**：使用多层神经网络的机器学习。

### 大语言模型

**LLM（大语言模型）**：基于 Transformer 的语言模型。

**Token**：文本被分割的基本单位。

**上下文窗口**：模型一次能处理的 Token 数量。

**提示词**：输入给模型的文本。

**幻觉**：模型生成的不准确或虚假信息。

### AI 开发

**Prompt Engineering**：提示工程，优化提示词。

**RAG（检索增强生成）**：结合检索和生成的 AI 系统。

**Fine-tuning**：在特定数据上微调预训练模型。

**Embedding**：将文本转为向量表示。

## 产品概念

### 产品开发

**MVP（最小可行产品）**：包含核心功能的最小版本。

**PMF（产品市场匹配）**：产品与市场需求匹配。

**A/B 测试**：对比两个版本的实验。

**用户旅程**：用户使用产品的完整路径。

### 商业模式

**B2B**：企业对企业。

**B2C**：企业对消费者。

**C2C**：消费者对消费者。

**SaaS（软件即服务）**：订阅制的软件服务。

**Freemium**：免费+增值服务的模式。

### 增长策略

**AARRR 漏斗**：获取、激活、留存、收入、推荐。

**病毒式传播**：用户自发传播产品。

**增长黑客**：以增长为目标的策略。

## 运营概念

### 用户运营

**DAU（日活跃用户）**：每日活跃用户数。

**MAU（月活跃用户）**：每月活跃用户数。

**留存率**：用户继续使用的比例。

**流失率**：用户停止使用的比例。

**LTV（生命周期价值）**：用户在整个生命周期贡献的价值。

**CAC（获客成本）**：获取一个用户的成本。

### 转化优化

**转化率**：完成目标行为的用户比例。

**漏斗分析**：分析转化过程中的流失。

**热力图**：用户行为的可视化。

### 内容运营

**SEO**：搜索引擎优化。

**内容营销**：通过内容获取用户。

**私域流量**：可反复触达的自己的用户。

## 安全概念

### Web 安全

**XSS（跨站脚本）**：注入恶意脚本。

**CSRF（跨站请求伪造）**：伪造用户请求。

**SQL 注入**：注入 SQL 语句。

**HTTPS**：安全的 HTTP 协议。

**CORS**：跨域资源共享。

### 数据安全

**加密**：将数据转为密文。

**哈希**：不可逆的数据转换。

**Token**：身份验证的令牌。

**OAuth**：开放的授权标准。

### 合规

**GDPR**：欧盟数据保护法规。

**CCPA**：加州消费者隐私法。

**等保**：中国信息安全等级保护。

## 项目管理概念

### 敏捷术语

**Sprint**：迭代周期，通常 1-4 周。

**Backlog**：待办列表，包含所有需求。

**Stand-up**：每日站会，同步进展。

**Retrospective**：回顾会议，总结改进。

**User Story**：用户故事，描述用户需求。

### 版本控制

**Git**：分布式版本控制系统。

**Branch**：分支，平行开发的版本。

**Merge**：合并分支。

**Pull Request**：请求合并代码。

**Conflict**：合并冲突。

### 文档

**README**：项目说明文档。

**API 文档**：接口文档。

**架构文档**：系统架构说明。

**变更日志**：版本更新记录。
