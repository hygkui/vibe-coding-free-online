# SEO 优化

## SEO 概述

SEO（Search Engine Optimization）是搜索引擎优化的缩写，指通过优化网站，提高在搜索引擎中的排名，从而获得更多自然流量。

对于产品网站来说，SEO 是低成本获取用户的重要渠道。

## SEO 基础

### 搜索引擎工作原理

**爬取**：搜索引擎蜘蛛抓取网页内容。

**索引**：将抓取的网页存入索引数据库。

**排名**：根据算法对搜索结果排序。

### 排名因素

**内容因素**：内容质量、关键词匹配、内容新鲜度。

**技术因素**：网站速度、移动适配、安全性。

**链接因素**：内部链接、外部链接。

**用户体验**：跳出率、停留时间、点击率。

### 白帽 vs 黑帽

**白帽 SEO**：遵循搜索引擎规则的合规优化。

**黑帽 SEO**：使用作弊手段，可能被惩罚。

## 技术 SEO

### 网站速度

**重要性**：速度影响用户体验和排名。

**优化方法**：压缩资源、使用 CDN、懒加载图片。

**测试工具**：PageSpeed Insights、Lighthouse。

### 移动适配

**响应式设计**：一个网站适配所有设备。

**移动优先**：Google 优先索引移动版本。

**测试工具**：Google Mobile-Friendly Test。

### 网站结构

**清晰的导航**：用户和搜索引擎都能轻松导航。

**URL 结构**：简洁、有意义、包含关键词。

```
推荐：/products/todo-app
不推荐：/p?id=12345
```

### HTTPS

**安全性**：搜索引擎偏好 HTTPS。

**证书**：使用 SSL 证书，Let's Encrypt 免费。

### 站点地图

**XML 站点地图**：列出网站所有页面，帮助搜索引擎发现内容。

**robots.txt**：告诉搜索引擎哪些页面可以抓取。

```xml
# robots.txt 示例
User-agent: *
Allow: /
Disallow: /admin/
Disallow: /private/

Sitemap: https://example.com/sitemap.xml
```

## 内容 SEO

### 关键词研究

**工具**：Google Keyword Planner、Ahrefs、SEMrush。

**选择原则**：搜索量适中、竞争度适中、与业务相关。

**长尾关键词**：更长、更具体的关键词，转化率高。

### 内容创作

**标题优化**：包含关键词，吸引点击。

```html
<title>待办事项管理工具 | 产品名称</title>
```

**Meta 描述**：总结页面内容，包含关键词。

```html
<meta name="description" content="高效的待办事项管理工具，帮助你规划任务、提高效率。免费试用。">
```

**内容结构**：使用标题标签（H1、H2、H3）组织内容。

**内容长度**：长内容通常排名更好（1000+ 字）。

### 内容更新

**定期更新**：保持内容新鲜。

**内容日历**：规划内容发布计划。

**更新旧内容**：提升现有内容的排名。

### 内容类型

**博客文章**：教育性内容、行业资讯。

**产品页面**：清晰的产品描述和优势。

**帮助文档**：解答用户问题，提升用户体验。

**案例研究**：展示成功案例，建立信任。

## 链接建设

### 内部链接

**链接相关页面**：帮助用户发现相关内容。

**锚文本**：使用描述性的锚文本。

```html
<a href="/features">功能介绍</a>
```

### 外部链接

**高质量链接**：来自权威网站的链接更有价值。

**获取方法**：创造高质量内容、媒体报道、合作伙伴。

### 避免的事情

**购买链接**：违反搜索引擎规则。

**垃圾链接**：低质量网站的链接。

**链接农场**：专门卖链接的网站。

## 本地 SEO

### 百度我的商家/Google My Business

对于有实体店铺的业务，创建商家信息。

### 本地关键词

包含地名和业务关键词，如"北京 待办事项应用"。

### 本地目录

在本地目录和黄页中列出业务。

## 技术工具

### 分析工具

**Google Analytics**：网站流量分析。

**Google Search Console**：搜索引擎表现监控。

**百度统计**：国内流量分析。

### SEO 工具

**Ahrefs**：反向链接分析、关键词研究。

**SEMrush**：全面的 SEO 分析工具。

**Moz**：SEO 工具和资源。

**Screaming Frog**：网站爬虫，检查技术 SEO 问题。

## 常见问题

### 新网站没有收录

检查 robots.txt 是否阻止抓取。提交站点地图到搜索引擎。创建高质量内容吸引爬虫。

### 排名下降

检查是否有算法更新。检查是否有负面 SEO 攻击。改进内容质量和用户体验。

### 关键词排名不理想

检查关键词竞争度。优化页面内容和结构。建设更多高质量链接。

## 电商产品 SEO 特别提示

### 产品页面优化

**产品标题**：包含品牌、型号、关键特征。

**产品描述**：详细、独特、包含关键词。

**图片优化**：使用描述性文件名和 alt 文本。

**评价**：鼓励用户留下评价，评价也是内容。

### 分类页面优化

**分类标题**：包含主要关键词。

**描述**：每个分类有独特的描述。

**筛选功能**：不影响搜索引擎抓取。

### 电商 SEO 策略

**长尾关键词**：更具体的购物意图关键词。优化产品评价页面：用户生成的内容。结构化数据：Schema.org 产品标记。
